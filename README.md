# ida_printk_str_recover

识别 Linux 内核风格 printk 字符串（以 0x01 + '0'..'7' 前缀），将 `.rodata` 中的字节序列自动转换为 IDA 字符串，并在数据项上添加可重复注释（去除日志级别前缀）。

## 安装

- 将 `plugins/printk_str_recover.py` 放入 IDA 插件目录，或直接在 IDA 中 `File -> Script file...` 运行。

## 使用

- 在菜单 `Edit` 下点击 `Recover printk strings (.rodata)` 运行。
- 插件会扫描 `.rodata*`/`__const`/`.rdata` 段，识别形如：
  - `0x01` + `'0'..'7'` + 文本 ... `\x00`
  - 如：`01 36 4d 65 64 69 61 20 ... 0x00`，会将其转为字符串，并在注释显示去前缀后的内容。

## 说明

- 仅处理以 `0x01+'0'..'7'` 开头、以 `\x00` 结尾的 ASCII 文本。
- 若已有定义会先删除对应项再设为 `C` 风格字符串。
- 注释设置为可重复注释，便于交叉引用查看。
